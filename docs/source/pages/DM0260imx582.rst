OAK-FFC IMX582
==============

`Buy it on Luxonis shop <https://shop.luxonis.com/products/oak-ffc-imx582>`__

.. thumbnail:: /_static/images/DM0260/oak-ffc-imx582.png

Overview
********

The **OAK-FFC IMX582** is a carrier board for the :ref:`IMX582` camera module. A 26-pin FFC is used
to carry 5V power, 4-lane MIPI, I2C, and other control signals between the this camera module and the controller baseboard.

Board Layout and Dimensions
***************************

* Width: 25 mm
* Height: 23.86 mm
* Weight: 3g

General information
*******************

* 26 pin 0.5mm FFC interconnect to baseboard
* 30 pin interface to IMX582 camera module
* 4-lane MIPI
* Aux/Control signals to camera module
* 5V power input via FFC
* On-board power generation for camera module
* Design files produced with Altium Designer 20
